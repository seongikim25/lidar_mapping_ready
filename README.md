# YDLIDAR G4 ROS2 Humble Driver

This repository is a ROS 2 Humble compatible port of the YDLIDAR G4 driver.

## Environment
- Ubuntu 22.04
- ROS 2 Humble
- YDLIDAR G4

## Changes from original
- Fixed rclcpp::declare_parameter usage for ROS2 Humble
- Refactored parameter declarations
- Tested with RViz2 (LaserScan)
